# Collins Muguro Gicheru - Portfolio Website

A modern, responsive portfolio website showcasing my journey and expertise in Food Science and Nutrition.

- CSS3
- Font Awesome Icons
- Modern CSS Features (Grid, Flexbox, Variables)
